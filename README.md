# Chip Inventory and Tracking System

## Project Overview
A web-based inventory management system for semiconductor manufacturing facilities to track chips, wafers, and related components with lot-level traceability.

## Academic Context
- Course: Software Engineering Lab
- Degree: B.Tech CSE (3rd Year)
- Development Methodology: Phase-wise SDLC (SRS → Design → Implementation → Testing → Deployment)

## Scope
- Single fabrication facility
- Role-based access (Admin, Operator, Auditor)
- Relational database-backed system
- Web-based UI

## Status
Project initiated. Requirements and design phases pending.
